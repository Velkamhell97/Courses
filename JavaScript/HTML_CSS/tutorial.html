CSS RULES

En este apartado se hablara sobre la reglas de jerarquia de los diferecntes tipos y selectores de css, asi como las demas formas de estilizar los elementos

Estilos de linea - Estos estilos son los que se aplican directamente dentro de la etiqueta html (1000 puntos)

IDs - En segundo lugar esta el selector ID, que en CSS se represeta por un # este selector tiene mayor peso que los demas selectores y combinaciones como clases . > ~ [atribute] etc, no obstante, tambien influye que tan especifico se declare este selector dentro del html, por ejemplo un div#a tiene mas peso que un #a (100 puntos)

Clases, Atributos y pseudo-clases - De tercer puesto se encuentran los selectores de clase, atributos y pseudo-clases . [atribute] :hover, :focus etc.. tambien influye que tan especifico se declare este selector dentro del html, por ejemplo un div.a tiene mas peso que un .a (10 puntos)

Elementos y pseudo-elementos - Por ultimo estan los selectores de etiqueta y los pseudo-elementos div a h1 ::before ::after etc.. (1 punto)

Este elemento tiene definido un estilo por medio de la etiqueta p tambien tiene un estilo mas especifico asignandole un ID al padre (div) #div-ID p or ultimo este elementotiene aplicado un estilo en la linea del html style="" siendo este de mayor orden jerarquico y sobreescribiendo a los demas

Igual especifidad: La ultima declaracion es la que cuenta - si se definene dos estilos que tengan el mismo nivel de jerarquia, por ejemplo definir un h2 y despues otro h2 el estilo que se aplicara sera el ultimo que se declare

Este p tiene dos estilos definido por la etiqueta p en el css #div-id p, la primera declaracion tiene un color de fondo rojo y la segunda tiene un color de fondo amarillo, al tener el mismo nivel de especifidad, el estilo que se aplica es el ultimo que se declara, en este caso el color de fondo es amarillo

El selector ID (#) tiene mas peso que el selector de atributo ([id=""]) - Esto se puede comprobar calculando el nivel de especifidad de cada elemento, acontinuacion se ahra un ejemplo

Este p tiene tres estilos definido por diferentes selectores, el primero es por el selector etiqueta-id p#p-id, el segundo selector solo esta definido por el id #p-id y el ultimo esta definido por el elemento[atributo] p[id=p-div], si hacemos el ejercicio de calcular notaremos que el primer estilo tiene un puntaje de 101 puntos el segudno estilo un valor de 100 puntos y el tercero 11 puntos esto indica que el estilo que se aplicara al p sera el primer estilo definido al tener mas valor de especifidad

Los estilos definidos dentro de un bloque style en el html tienen mas peso que los definidos en el CSS - esto es similar al caso de los estilos en linea, los declarados dentro del bloque style en el html tienen mas peso que los del css

el estilo de este p esta definido tanto en el css como dentro del bloquye de estilos en el html por medio del id del padre #div-id p en este caso el estilo que se aplica es el declarado en el bloque style="" esto solo aplica para cuando los elementos tienen el mismo nivel de especifidad

Este p al igual que el anterior tiene 2 estilos definidos uno en el css y otro en el bloque html style, no obstante a pesar de que la jerarquia del bloque style es mayor, el nivel de especifidad del p en el css es mayor, por lo tanto se aplcara este estilo
